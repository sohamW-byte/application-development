# Todo List (React Native / Expo)

Minimal todo list built with React Native and Expo.

Requirements
- Node.js and npm
- Expo CLI (optional; you can use npx expo)

Install

Open PowerShell in the project directory and run:

```powershell
npm install
# or: npx expo install
```

Run

```powershell
npx expo start
# then press 'a' to open on Android emulator or scan the QR with Expo Go on your device
```

Files
- `App.js` - main app with add/toggle/delete logic
- `components/TodoItem.js` - small presentational component
